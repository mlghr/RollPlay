# Roll Play - Full stack applicaton with API

## High level features:

1) Frontend
2) Backend
3) API functionality (http verbs)

## INSTRUCTIONS TO START APP

1) Clone this repo
2) `cd rollplay`
3) `npm i`
4) `cd rollplay/backend`
5) `npm i`
6) `cd rollplay/frontend`
7) `npm i`
8) Navigate to localhost:3000
